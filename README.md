### Завдання 2
##### Запуск
1. Перейдіть у папку task_2
2. Запустіть команду:
```bash
docker-compose up
```
##### Результат
В браузері відкрийте сторінку застосунку за адресою:
```
http://localhost:8080
```
Настисніть на кнопку "Перевіпити БД".

Якщо БД працює, ви побачите відповідне повідомлення:
```
"Welcome to FastAPI!"
```
